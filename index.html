What is Function ?

It is a piece of code which will be used for reusability, it means write once , call multiple times or call anywhere

Function Types

I. Function without parameters, without return. 

                function add(){
                    console.log('Hi')
                }
                add();  //Hi

II. Function with params, without return. 

                function name(userName){
                    console.log(userName);
                }
                name('Deepak'); //Deepak

III. Function with params, with return. 

                function multiply(a,b){
                    return a*b;
                }
                console.log(multiply(3,5)); //15
